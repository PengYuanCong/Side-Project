# Side-Project

please download proposal.rar
it's the side project about crud function with MVC。

🚀技術棧 (Tech Stack)Backend: C# / ASP.NET Core MVC (8.0) Database: SQL Server (SSMS) / ADO.NET / LINQ Frontend: HTML5 / CSS3 / JavaScript / Bootstrap 5 / AJAX Libraries: ClosedXML (Excel 處理) / FullCalendar (行事曆) 
🌟 核心功能 (Key Features)1. 會員系統與安全性身份驗證： 實作 Cookie-based 驗證機制。 權限管理： 具備角色權限控管 (RBAC)，確保一般用戶與管理員看到不同的介面。 資料隔離： 完善的會員資料隔離，確保使用者只能存取自己的數據。 2. 裝備管理庫 (CRUD)屬性管理： 提供完整的裝備增刪查改功能。 
Excel 整合： * 支援 Excel 快速匯入，一鍵建立大量裝備資料。 支援 Excel 報表匯出，方便線下儲存或分享。 3. 智慧計算與紀錄CP 值公式計算： 內建演算法，根據裝備價格與屬性加總，自動計算性價比。 計算紀錄回溯： 系統會自動保存最近的計算紀錄，方便隨時查看。 4. 個人化行程管理動態行事曆： 整合 FullCalendar，讓使用者紀錄重要任務或遊戲事件。
